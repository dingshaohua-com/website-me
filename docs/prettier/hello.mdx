## 安装

```shell
pnpm install --d prettier
```


## 配置文件

创建一个配置文件，如下
```js title="prettier.config.js"
module.exports = {
  tabWidth: 2, // 缩进2个空格
  useTabs: false, // 缩进单位是否使用tab替代空格
  semi: true, // 句尾添加分号
  singleQuote: true, // 使用单引号代替双引号
};
```

## 使用
```shell
npx prettier --write .
```
