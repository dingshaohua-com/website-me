在2012 年国庆节这天，微软首次将TypeScript 公开亮相，其创始人`安德斯·海尔斯伯格` 这样描述它:
> 它是 JavaScript 的类型化超集，可被编译成常用的 JavaScript。它 还可以通过启用丰富的工具体验来极大地帮助提升生产力，与此同时开发者保持不变维护现有的代码，并继续使用喜爱的 JavaScript 库。
