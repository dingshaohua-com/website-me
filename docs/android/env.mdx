## 安装 android sdk
sdk即软件开发工具包。   
这是安装完android Studio 要做的第一件事情  
相当于：后端开发 装完了idea 还需要装java环境，前端开发 装完vscode 还需要装node环境 一个道理！


况且当你第一次打开android Studio 它就会提示你必须安装sdk才能继续

## 后续管理 android sdk
如果你的项目sdk（android sdk platform）和你运行的设备sdk不一致 会弹窗提示 如果点击继续 则和可能运行失败！   
这种情况会出现在你导入已有的项目，或者你的运行设备更换了 比如换手机了。  
<img src={useBaseUrl("/img/android/20230727114438.png")} width="600" />

这个时候 你就需要来管理sdk了，可以在以下两个地方找到入口
<img src={useBaseUrl("/img/android/20230727114436.png")} width="600" />      
<img src={useBaseUrl("/img/android/20230727114437.png")} width="600" />

另外新版的 android Studio 做了一件伟大的事情：   
如果你创建新项目的时候 如果没有对应版本的sdk，它会帮你自动下载。   
所以基本上除非特殊情况，否则我们不需要在去手动去维护sdk了！      
更不要sdk焦虑，提前下好一堆sdk！
